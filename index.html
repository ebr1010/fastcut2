⚙ GitHub 同期設定

各デバイスで1回だけ設定してください。
同じリポジトリ・同じファイルパス・同じTokenを入力し「保存して同期」を押すと、このデバイスの設定が保存され即座にデータを同期します。重複登録の心配はありません（差分マージ方式）。
設定済みのまま最新データを取り込みたい場合は「今すぐ同期」を押してください。
